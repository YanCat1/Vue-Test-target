复合选择器:

  1. 后代选择器 空格

     根据HTML标签的嵌套关系，选择父元素 后代中满足条件元素

     选择器语法: 选择器1 选择器2{ css样式 } 

     后代选择器中，选择器之间使用空格进行分隔。

2. 子代选择器  >

   语法：选择器 > 选择器{ css }

   注意 只能找到儿子，之间使用 > 隔开

3. 并集选择器  ,

   语法：选择器1 ， 选择器2{css}
